# InputComponent

Este é um componente de entrada de texto para formulários.

## Props
<table style={{ width: '100%' }}>
  <thead>
    <tr>
      <th style={{ width: '20%' }}>Nome</th>
      <th style={{ width: '20%' }}>Tipo</th>
      <th style={{ width: '20%' }}>Obrigatório</th>
      <th style={{ width: '40%' }}>Descrição</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>label</td>
      <td>String</td>
      <td>Sim</td>
      <td>Texto que descreve o campo de entrada</td>
    </tr>
    <tr>
      <td>placeholder</td>
      <td>String</td>
      <td>Sim</td>
      <td>Texto que aparece como espaço reservado no campo de entrada</td>
    </tr>
    <tr>
      <td>name</td>
      <td>String</td>
      <td>Sim</td>
      <td>Texto referente ao nome do input</td>
    </tr>
    <tr>
      <td>type</td>
      <td>String</td>
      <td>Sim</td>
      <td>Texto referente ao tipo de input (email, password, text, etc)</td>
    </tr>
    <tr>
      <td>downLabel</td>
      <td>Object</td>
      <td>Não</td>
      <td>Objeto com as propriedades `message` (texto do rótulo abaixo do campo de entrada), `isOnTheRight` (booleano que indica se o rótulo abaixo do campo de entrada deve estar à direita) e `smartLinkPath` (string que indica o caminho do smartLink abaixo do input. Não é obrigatório)</td>
    </tr>
    <tr>
      <td>inputValidation</td>
      <td>Object</td>
      <td>Não</td>
      <td>Objeto com as propriedades `isInvalid` (booleano que indica se o input esta inválido) e `message` (texto de erro a ser exibido abaixo do input. Caso a message seja undefined o valor default será "Campo obrigatório").</td>
    </tr>
  </tbody>
</table>
