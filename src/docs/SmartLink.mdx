# SmartLink

Este é um componente de link que pode ser personalizado com cores e efeitos de sublinhado.

## Props

      <td>O texto que aparece dentro do link. Se não for fornecido, o valor padrão é 'Label'.</td>
    </tr>
    <tr>
      <td>underline</td>
      <td>Boolean</td>
      <td>Não</td>
      <td>Define se o link deve ter sublinhado ou não. O valor padrão é 'false'.</td>
    </tr>
    <tr>
      <td>primaryPink</td>
      <td>Boolean</td>
      <td>Não</td>
      <td>Define se o link deve ter a cor primária de rosa ou não. O valor padrão é 'false'.</td>
    </tr>
    <tr>
      <td>href</td>
      <td>String</td>
