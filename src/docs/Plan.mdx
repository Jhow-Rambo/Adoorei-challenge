# PlanComponent

Este é o componente que exibe informações sobre os planos oferecidos pelo nosso serviço.

## Props

<table style={{ width: '100%' }}>

    </tr>
  </thead>
  <tbody>
    <tr>
      <td>planName</td>
      <td>String</td>
      <td>Sim</td>
      <td>Nome do Plano</td>
    </tr>
    <tr>
      <td>price</td>
      <td>Number</td>
      <td>Não</td>
      <td>Preço do plano, caso não seja passado nada será atribuido Grátis ao plano</td>
    </tr>
    <tr>
      <td>description</td>
      <td>String</td>
      <td>Sim</td>
      <td>Descrição do plano</td>
    </tr>
    <tr>
      <td>idealFor</td>
      <td>String</td>

    </tr>
  </tbody>
</table>

## Exemplo de uso

```jsx
<PlanComponent
  planName="Plano Standard"
  price={100}
  description="Descrição do Plano Standard"
  idealFor="Ideal para pessoas físicas"
  planIndex={0}
  isSelectedPlan={true}
  isMostUsedPlan={true}
/>
